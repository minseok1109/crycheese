# 크라이치즈버거 홈페이지 디자인 - Option C: 플레이풀 맥시멀리즘

- 작성일: 2026-01-22
- 상태: 비교 검토용
- 미학적 방향: **Playful Maximalism (Fun & Memorable)**

## 컨셉 개요

### 톤 & 무드
- 과감하고 재미있는 맥시멀리스트 디자인
- 스티커, 배지, 손글씨 요소
- 브랜드 캐릭터 '엉엉이' 적극 활용
- 기억에 남는 유니크한 경험

### 핵심 차별점
> **"버거 주문이 이렇게 즐거울 수 있다고?"**
> 딱딱한 B2B가 아닌, 친근하고 재미있는 경험

## Animation & Interaction

### 바운스 효과
```tsx
gsap.to(".bounce-btn", {
  y: -5,
  duration: 0.5,
  ease: "power2.out",
  yoyo: true,
  repeat: -1
});
```

### 흔들림 효과
```tsx
gsap.to(".wiggle", {
  rotation: 3,
  duration: 0.3,
  ease: "power1.inOut",
  yoyo: true,
  repeat: -1
});
```

### 스티커 등장
```tsx
gsap.from(".sticker", {
  scale: 0,
  rotation: -20,
  duration: 0.5,
  ease: "back.out(2)",
  stagger: 0.1
});
```

### 캐릭터 반응
```tsx
// 스크롤 시 엉엉이가 따라다님
gsap.to(".character", {
  scrollTrigger: {
    trigger: ".section",
    scrub: true
  },
  y: 100
});

// 호버 시 반응
gsap.to(".character", {
  scale: 1.1,
  duration: 0.3,
  ease: "back.out"
});
```

### 인풋 포커스
```css
.input-playful:focus {
  outline: none;
  border-color: var(--color-yellow);
  box-shadow:
    0 0 0 4px var(--color-yellow-light),
    4px 4px 0 var(--color-black);
  transform: translateY(-2px);
}
```

| 말풍선 | 중 | CSS :after pseudo |
