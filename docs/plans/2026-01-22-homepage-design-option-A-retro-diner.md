# 크라이치즈버거 홈페이지 디자인 - Option A: 레트로 다이너

- 작성일: 2026-01-22
- 상태: 비교 검토용
- 미학적 방향: **Retro Diner (1950s American Diner)**

---

## 컨셉 개요

### 톤 & 무드
- 1950년대 미국 다이너의 따뜻하고 향수를 불러일으키는 감성
- 네온 사인, 체크 패턴, 크롬 디테일
- 클래식하면서도 신뢰감 있는 B2B 느낌

### 핵심 차별점
> **"타임머신을 타고 돌아온 정통 아메리칸 버거의 맛"**
> 레트로 감성이 주는 친근함과 검증된 맛에 대한 신뢰

## Animation & Interaction

### 네온 효과
```tsx
// 네온사인 깜빡임
const neonFlicker = gsap.timeline({ repeat: -1 });
neonFlicker
  .to(".neon-text", { opacity: 1, duration: 0.1 })
  .to(".neon-text", { opacity: 0.8, duration: 0.1 })
  .to(".neon-text", { opacity: 1, duration: 0.1 })
  .to(".neon-text", { opacity: 0.9, duration: 0.05 })
  .to(".neon-text", { opacity: 1, duration: 2 });
```

### 버거 회전
```tsx
gsap.to(".burger-hero", {
  rotation: 5,
  y: -10,
  duration: 2,
  ease: "power1.inOut",
  yoyo: true,
  repeat: -1
});
```

### 폴라로이드 스캐터
```tsx
gsap.from(".polaroid", {
  rotation: "random(-15, 15)",
  scale: 0.8,
  opacity: 0,
  stagger: 0.2,
  ease: "back.out(1.7)"
});
```
