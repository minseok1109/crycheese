# 메뉴 섹션 리디자인

## 개요

메뉴 프리뷰 섹션의 레이아웃 개선. 현재 14개 메뉴가 4개 카테고리에 나열되어 세로로 길고 반복적인 느낌을 주는 문제 해결.

## 목표

- 전체 메뉴를 보여주면서 다양성 강조
- 세로 길이 감소 (스크롤 최소화)
- 카테고리별 반복 느낌 제거
- 하위 메뉴 카드 크기 개선

## 디자인

### 전체 구조

```
┌─────────────────────────────────────────────────┐
│  메뉴 구성 (타이틀)                               │
├─────────────────────────────────────────────────┤
│  RECOMMENDED                                     │
│  ───────────                                     │
│  [큰 카드 3개 - 기존 유지]                        │
├─────────────────────────────────────────────────┤
│  BURGER    SIDE    SET       ← 탭 네비게이션     │
│  ═══════                     (활성: 노란 밑줄)   │
│                                                  │
│  [선택된 카테고리의 메뉴만 표시]                  │
└─────────────────────────────────────────────────┘
```

### 탭별 콘텐츠 레이아웃

**BURGER 탭 (주력 상품)**
- 카드 크기: 280px 너비, 200px 이미지 높이 (추천 메뉴와 동일)
- 배치: 3개씩 한 줄
- 4개 아이템: 싱글치즈버거, 더블치즈버거, 치킨샌드위치, 불치킨샌드위치

**SIDE 탭 (보조 상품)**
- 카드 크기: 200px 너비, 140px 이미지 높이
- 배치: 4개씩 한 줄
- 3개 아이템: 크라이치킨접시, 치즈감자, 치킨핑거

**SET 탭 (보조 상품)**
- 카드 크기: 200px 너비, 140px 이미지 높이
- 배치: 4개씩 한 줄
- 4개 아이템: 싱글치즈버거세트, 더블치즈버거세트, 치킨샌드위치세트, 불치킨샌드위치세트

### 탭 UI 스타일링

```css
/* 탭 스타일 */
비활성 탭: color #999999, 밑줄 없음
활성 탭: color #0D0D0D, 밑줄 #F8B81C (2px)
호버: color #0D0D0D
폰트: 12px, font-weight 600, letter-spacing 2px
탭 간격: 32px
탭 영역 아래 구분선: #E5E5E5
```

- 기본 선택: BURGER

### 애니메이션

- 기존 추천 메뉴 GSAP 애니메이션 유지
- 탭 전환: fade-in (opacity 0→1, y 20→0, 300ms)
- 카드 stagger: 0.1초 간격

### 반응형 (모바일)

- 탭: 한 줄에 3개 모두 표시 (간격 축소)
- BURGER 카드: 2열 그리드
- SIDE/SET 카드: 2열 그리드

## 기대 효과

| 항목 | 현재 | 개선안 |
|------|------|--------|
| 세로 길이 | 4개 섹션 풀 표시 | 1개 탭 콘텐츠만 표시 (~60% 감소) |
| 카테고리 구분 | 반복적인 레이아웃 | 탭 네비게이션으로 전환 |
| 하위 메뉴 크기 | 모두 200px 동일 | BURGER 280px / SIDE,SET 200px |
| 기본 표시 | 모든 카테고리 | 추천 + BURGER 탭 |
