## 컨셉 개요

### 톤 & 무드
- 깔끔하고 세련된 프리미엄 푸드 브랜드
- 미니멀리즘 + 강렬한 포인트 컬러
- 기업 고객에게 신뢰감과 전문성 전달

### 핵심 차별점
> **"프리미엄 케이터링의 새로운 기준"**
> 깔끔한 UX와 정돈된 정보로 신뢰를 주는 B2B 서비스

## Animation & Interaction

### 페이지 로드
```tsx
// 순차적 등장 (stagger)
gsap.from(".fade-up", {
  y: 40,
  opacity: 0,
  duration: 0.8,
  stagger: 0.1,
  ease: "power3.out"
});
```

### 스크롤 트리거
```tsx
gsap.from(".section-content", {
  scrollTrigger: {
    trigger: ".section",
    start: "top 80%",
  },
  y: 60,
  opacity: 0,
  duration: 1,
  ease: "power3.out"
});
```

### 호버 인터랙션
```css
/* 카드 호버 */
.card {
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}
.card:hover {
  transform: translateY(-4px);
  box-shadow: var(--shadow-elevated);
}

/* 버튼 호버 */
.btn-primary {
  transition: background-color 0.2s ease, transform 0.2s ease;
}
.btn-primary:hover {
  background-color: var(--color-primary-dark);
  transform: scale(1.02);
}
```

### 로고 그레이스케일
```css
.client-logo {
  filter: grayscale(100%);
  opacity: 0.6;
  transition: all 0.3s ease;
}
.client-logo:hover {
  filter: grayscale(0%);
  opacity: 1;
}
```
