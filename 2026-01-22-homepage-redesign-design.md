## 개요

크라이치즈버거 기업 케이터링 서비스 랜딩 페이지 리디자인. 기존 요구사항(`requirements/group-order-requirements.md`)을 폐기하고 새 계획서(`홈페이지 작업 계획서.md`)를 기준으로 진행.

## 작업 범위

- **프론트엔드**: UI 섹션 전체 (기존 컴포넌트 수정 + 신규 3개)
- **백엔드**: Google Sheets + Slack 연동 (환경변수 기반)
- **이미지**: 플레이스홀더 사용 (추후 실제 이미지 교체)

### 9. InquiryForm (기존 수정)

**주문자 정보**
| 필드 | 타입 | 필수 |
|------|------|------|
| 이름 | text | ✓ |
| 연락처 | tel | ✓ |
| 이메일 | email | ✓ |
| 유입경로 | radio | - |

유입경로 옵션: 포털검색, 재주문, 지인추천(추천자명 입력), 블로그/카페, 매장방문

**조직 정보**
| 필드 | 타입 | 필수 |
|------|------|------|
| 단체명 | text | ✓ |
| 행사유형 | select | - |

**주문 정보**
| 필드 | 타입 | 필수 |
|------|------|------|
| 배송 희망 날짜 | date | ✓ |
| 배송 희망 시간 | time | ✓ |
| 배송 상세 주소 | text | ✓ |
| 결제 희망 방법 | radio | ✓ |

결제 옵션: 계좌이체, 카드결제

**문의 내용**
| 필드 | 타입 | 필수 |
|------|------|------|
| 문의글 | textarea | - |

## 백엔드 API

### POST /api/inquiry

```
입력값 검증 (Zod)
       │
       ├──▶ Google Sheets 저장 (옵셔널)
       │
       └──▶ Slack 알림 전송 (옵셔널)
       │
       ▼
응답 반환 → /inquiry/complete 리다이렉트
```

**환경변수**
```env
SLACK_WEBHOOK_URL=https://hooks.slack.com/services/...
GOOGLE_SHEETS_ID=1abc...xyz
GOOGLE_SERVICE_ACCOUNT_KEY={"type":"service_account",...}
```

**Slack 메시지 포맷**
```
📋 새 주문 접수

• 담당자: {name}
• 단체명: {companyName}
• 배송일시: {date} {time}
• 주소: {address}
• 연락처: {phone}
• 결제방법: {paymentMethod}

💬 문의 내용:
{message}
```
